# 📖 Overview

AEGIS AI is an Enterprise Industrial Intelligence Platform developed for the **ET AI Hackathon**.

Instead of treating industrial safety as isolated alarms, AEGIS AI continuously correlates multiple operational signals to detect **compound risks** before they escalate into accidents.

The platform combines:

- 🏭 Industrial Digital Twin
- 🤖 Multi-Agent AI
- 🧠 Explainable AI
- 📊 Executive Intelligence
- 📡 Real-Time Sensor Simulation
- 📚 RAG-Based Compliance Assistant
- 🚨 Emergency Orchestration

to provide proactive operational awareness for safety-critical industries.

---

# 🚀 Problem Statement

Traditional industrial monitoring systems generate thousands of independent alarms but fail to understand relationships between operational events.

A seemingly harmless increase in gas concentration combined with:

- Active Hot Work
- Worker Presence
- Equipment Maintenance
- Pressure Increase

may become a catastrophic incident.

AEGIS AI detects these **compound hazards** before conventional monitoring systems.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/aegis-ai.git

cd aegis-ai
```

Install dependencies

```bash
npm install
```

---

# 🔑 Environment Variables

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

# ▶️ Running the Project

Frontend

```bash
npm run dev
```

Backend

```bash
npm run server
```

Open

```
http://localhost:5173
```
